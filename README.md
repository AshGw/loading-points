 
## Installation 
**npm**
```bash
npm i ts-roids
```
**pnpm**
```bash
pnpm i ts-roids
```

## Usage 
```tsx
import Loading from 'loading-points';

function LoadingScreen() {
  return (
    <div className="flex items-center justify-center h-screen">
      <Loading glowColor='rgb(155, 46, 199)' circleSize='8px' />
    </div>
  );
}

export default LoadingScreen;
```
